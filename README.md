<h2>Learning Management System (LMS)</h2>

<p>
This project is a <strong>role-based Learning Management System (LMS)</strong> designed to manage online learning activities for an institute.
The system provides different levels of access and functionality based on user roles to ensure secure and organized learning workflows.
</p>

<h3>Role-Based Access</h3>

<p>
The LMS supports three primary roles:
<strong>Admin</strong>, <strong>Mentor</strong>, and <strong>Student</strong>.
Each role has specific permissions and responsibilities within the system.
</p>

<h4>Admin</h4>
<ul>
  <li>Manage and create course categories</li>
  <li>View all registered users</li>
  <li>Edit and update user information</li>
  <li>Maintain overall system control and data integrity</li>
</ul>

<h4>Mentor</h4>
<ul>
  <li>Create new courses</li>
  <li>Update and manage existing courses</li>
  <li>View enrolled students for a particular course</li>
  <li>Manage course content and assignments</li>
</ul>

<h4>Student</h4>
<ul>
  <li>Browse and enroll in available courses</li>
  <li>Access enrolled course content</li>
  <li>Submit assignments related to enrolled courses</li>
  <li>Track learning progress</li>
</ul>

<h3>Key Features</h3>
<ul>
  <li>Secure authentication and authorization</li>
  <li>Role-based access control (RBAC)</li>
  <li>Course and category management</li>
  <li>Assignment submission and enrollment tracking</li>
  <li>Scalable architecture suitable for institutional use</li>
</ul>

<p>
This LMS is built as a full-stack application and is suitable for academic institutions looking to manage courses, mentors, and students in a structured and secure manner.
</p>
